# Logo NGTC

## Instruksi

Simpan logo **NGTC (Nagatech Sistem Integrator)** di folder ini dengan nama:

```
ngtc-logo.png
```

### Spesifikasi Logo:
- **Format**: PNG (dengan background transparan lebih baik)
- **Ukuran**: Tinggi 50-100px (width auto)
- **Nama File**: `ngtc-logo.png`
- **Lokasi**: `static/images/ngtc-logo.png`

### Langkah-langkah:
1. Pastikan Anda memiliki file logo NGTC
2. Rename file menjadi `ngtc-logo.png`
3. Copy file ke folder ini: `C:\POS\AppServ\www\NagaCageur\static\images\`
4. Refresh browser untuk melihat logo muncul di header

### Catatan:
Jika logo tidak ada, aplikasi akan tetap berjalan normal tanpa menampilkan logo (tidak akan error).

Logo akan muncul di bagian header aplikasi dengan background putih dan shadow effect untuk tampilan yang lebih profesional.
